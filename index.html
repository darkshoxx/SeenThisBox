THE SITUATION

Somewhere in my memory lives a video game. I cannot tell you its name — if I knew that, we wouldn't be here. I received it as part of a box of PC games one Christmas, somewhere between 2003 and 2007. The bundle included the original Rhem — a German-made point-and-click — and about eight to ten other games, all obscure, all in their own shrink-wrapped cardboard boxes.

One of those games had an object in it I have never forgotten: a huge black box, roughly the size of a supermarket freezer, with words written on it in red. Something like "Red Herring Detector", or "Anti-Red-Herring Device" — the exact name is one of the things I'm trying to recover.

The mechanic was simple and brilliant. The game knew which inventory items you no longer needed for progression. Feed one into the box, and it would vanish. A device for clearing the dead ends from your pockets. I found it genuinely clever, which is exactly why it stuck.

I've been hunting for this game long enough that I built a physical replica of the box from memory. If you recognise any of this — the box, the mechanic, the game — I need to hear from you.

WHAT I REMEMBER
  • ▸ Platform: PC, Windows XP / 2000 / ME era. Almost certainly not a console release. Mac not ruled out.
  • ▸ Estimated release: 2003–2007. Received before 2007, on the same Christmas as the original Rhem (released in Germany, May 2003).
  • ▸ Genre: First-person point-and-click adventure. Static frame navigation — no character animations. In the style of Myst or Rhem.
  • ▸ Graphics: Low-budget 3D renders. Dark colour palette — possibly set at night. Indie feel; I remember being a little disappointed by the production quality at the time.
  • ▸ Inventory UI: White-outlined squares as item placeholders; UI was predominantly black-filled. Items displayed as icons in a grid.
  • ▸ The object: A huge black box — roughly freezer-sized — with red text reading something like "Red Herring [X]", where X is a word like Box, Detector, Device, or similar. Located inside a building — a house or hut.
  • ▸ The mechanic: Items fed into the box that were no longer needed for progression were permanently removed from your inventory.
  • ▸ Language: The box text was in English, which may suggest it wasn't a German studio — though this is uncertain.
  • ▸ Bundle context: Part of an informal collection of ~10 obscure/indie PC games. Each game came in its own original cardboard box, apparently shrink-wrapped — not second-hand, and not an official bundle.

These are reconstructed memories. Some details may be inexact or conflated. If something is "almost" right — still reach out. Close is useful.
